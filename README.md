# Inventory and Sales Management System 

A desktop-based Inventory and Sales Management System developed for small retail stores to automate inventory tracking, sales processing, and customer management.
This is my very first and a simple management system developed for the ICT Project Module during the first year of my degree program.

## Features
- Product Management
- Customer Management
- Order Processing
- Stock Tracking
- Sales Reports
- Secure Login System

## Technologies Used
- C#
- Windows Forms
- SQL Server
- .NET Framework
- T-SQL

## Modules
1. Login Module
2. Home Dashboard Module
3. Category Management Module
4. Customer Management Module
5. Product Management Module
6. Order Management Module
7. View Orders Module
8. User Management Module
9. Splash Screen Module

## System Functions
- Add, Edit, Delete, and View Products
- Manage Customers and Categories
- Process Orders and Sales
- Automatic Stock Updates
- Generate Reports
- User Authentication and Authorization
- Store Transaction History
- Sales Total Calculation
- Input Validation and Error Handling
